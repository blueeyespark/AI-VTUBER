# AI Blue Data Center

The data center consolidates the Project Blue foundation draft and 18 source screenshots captured on July 1, 2026.

## Directory map

| Directory | Purpose |
|---|---|
| `00_START_HERE` | Orientation, index, terminology, and source register |
| `01_FOUNDATION` | Mission, constitution, identity, and legacy |
| `02_ARCHITECTURE` | System boundaries, modules, memory, and data |
| `03_CAPABILITIES` | Creation, presence, knowledge, enterprise, and robotics |
| `04_SECURITY_GOVERNANCE` | Security, permissions, continuity, safety, and stewardship |
| `05_ENGINEERING` | Requirements, interfaces, testing, and implementation rules |
| `06_OPERATIONS` | Deployment, migration, monitoring, backup, and recovery |
| `07_ROADMAP` | Phases, decisions, risks, and open questions |
| `08_SOURCE_ARCHIVE` | Catalog and provenance for original source materials |

## Authority order

When documents conflict, resolve them in this order:

1. Blue Constitution
2. Safety, privacy, consent, and applicable law
3. Approved governance decisions
4. Engineering requirements
5. Capability proposals
6. Historical source material

Historical screenshots preserve intent, but they are not automatically executable requirements.
